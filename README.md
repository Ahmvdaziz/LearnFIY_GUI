# LearnFIY_GUI

Simple static demo of a learning platform UI (landing + auth pages).

## Quick start

- Open `index.html` in your browser to view the landing page.
- Auth page: `pages/signup-login.html`.

## Project structure

- `index.html` — landing page
- `pages/signup-login.html` — login & signup UI
- `assets/css/` — styles (`styles.css`, `auth.css`)
- `assets/js/` — scripts (`script.js`, `auth.js`)
- `assets/js/` — scripts (`script.js`, `auth.js`, `flashcards.js`)

## Notes

- This is a static front-end mockup (no backend). Forms use client-side validation only.
- Feel free to ask me to add a simple dev server or GitHub Actions workflow.

## New: Flashcards

- A simple flashcards demo was added: `pages/flashcards.html` with supporting CSS/JS under `assets/css/flashcards.css` and `assets/js/flashcards.js`.

You can open `pages/flashcards.html` to try the feature locally.
